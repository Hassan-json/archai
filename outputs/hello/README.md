# Hello World App

## Overview
A simple Hello World application built with Flask. 

## Requirements
- Python 3.x
- Flask

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Run the application: `python main.py`
3. Access the API at `http://127.0.0.1:5000/hello`

## Endpoint
- **GET /hello**: Returns a JSON response with the message "Hello, World!"
